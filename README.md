# HR Analytics Dashboard | Microsoft Excel

## Overview
This project analyzes an HR dataset containing 1,470 employee records using Microsoft Excel. The dashboard provides interactive insights into workforce distribution, employee attrition, salaries, and years at the company.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Monthly Income
- Average Years at Company

## Dashboard Insights
- Department-wise Employee Distribution
- Gender Distribution
- Attrition by Department
- Salary Analysis
- Years at Company Analysis

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Development
- Data Visualization
- KPI Reporting
- Business Analysis

## Dataset
IBM HR Analytics Employee Attrition Dataset (1,470 Records)
